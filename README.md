# Java 25 + Spring AI Starter

Educational starter template by **Garima Agarwal** (@agarwal24garima)  
**Day 1/30** of consistent GitHub push series.

### Features
- Java 21 baseline + Java 25 compact source & modern features
- Spring Boot 4.0.6
- Spring AI 1.1.5 with local Ollama (Mistral)
- Virtual Threads ready
- Clean architecture for learning

### Quick Start
1. Install Ollama → `ollama run mistral`
2. `mvn spring-boot:run`
3. Open browser: http://localhost:8080/ai/hello
